Skip-Def Diagnostics (C# side check)
-------------------------------------
  ALL 1048 IR defs classified as 'Non-lowercase' in C# diagnostics.
  This is a CCE encoding artifact: def names are stored in CCE format
  where character codes differ from ASCII. The C# check compared against
  ASCII 'a'(0x61)-'z'(0x7A), but CCE uses different code points.
  The self-hosted skip-def uses char-code-at which operates on CCE codes.

Prime Suspects
--------------
  1. skip-def (CodexEmitter.codex:114) — filters by ctor-names, empty name,
     first-char range, and is-error-body. The first-char range check uses
     CCE char codes; verify the range covers all lowercase CCE codes.
  2. emit-defs-streaming (main.codex:128) — the streaming pipeline that calls
     emit-def and skips defs where result is empty string.
